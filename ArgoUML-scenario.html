ArgoUML-scenario

  1. Launch the application.

  2. Select the class figure placement mode in the toolbar and place a class figure in the edit area
    (the feature PLACE CLASS FIGURE).

  3. Unselect the class figure placed at 2.
  4. Select the selection mode in the toolbar and click the class figure to select
    (the feature SELECT FIGURE).

  5. Pressed ‘Del’ key to remove the class figure (the feature DELETE FIGURE).

  6. Click the close button in the application window to close the application.
