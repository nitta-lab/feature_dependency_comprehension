JHotDraw-scenario

  1. Launch the application.
  2. Select the rectangle figure placement mode in the toolbar and place a rectangle figure in the edit area
    (the feature PLACE RECTANGLE FIGURE).
  3. Unselect the rectangle figure placed at 2.
  4. Select the selection mode in the toolbar and click the rectangle figure to select
    (the feature SELECT FIGURE).
  5. Drag the figure and drop it in the edit area to move the figure
    (the feature TRANSFER FIGURE).
  6. Click the close button in the application window to close the application.
